###FFmpeg常用命令分类

#### 基本查询命令

| 命令         |                                            |
| ------------ | ------------------------------------------ |
| -version     | 版本                                       |
| -demuxers    | 可以使用的解封装  分解                     |
| -muxers      | 可以使用的复用      复用                   |
| -devices     | 输入设备                                   |
| -codecs      | 所有的编码/解码器                          |
| -decoders    | 可用解码器                                 |
| -encoders    | 可用编码器                                 |
| -bsfs        | 显示可用的比特流filter  (对比特流进行处理) |
| -formats     | 显示可用的格式                             |
| -protocols   | 输入文件的协议查询(rtmp等等)               |
| -filters     | 所有可用的fiter                            |
| -pix_fmts    | 像素格式  rgb  yuv                         |
| -sample_fmts | 采样格式(位数 float)                       |
| -layouts     | 显示channel名称                            |
| -colors      | 显示支持的颜色                             |
